# 🚀 Oracle Database Development Handbook

> **Build production-ready Oracle SQL & PL/SQL code that follows enterprise development standards from day one.**

**Oracle Database Development Handbook** is a **Claude Skill** that helps developers generate Oracle SQL & PL/SQL objects using enterprise coding standards, reusable templates, documentation headers, validations, and best practices.

Instead of repeatedly writing boilerplate code, simply describe what you want to build, and Claude generates production-ready Oracle database objects.

Whether you're a beginner, an experienced Oracle developer, or an enterprise development team, this skill helps you write cleaner, more maintainable code while saving valuable development time.

---

# ✨ Why I Built This

After spending **16+ years as an Oracle Developer**, I realized something.

Knowing SQL or PL/SQL syntax is only a small part of the job.

The real challenge is making sure every object follows:

- Company coding standards
- Documentation headers
- Naming conventions
- Exception handling
- Validations
- Best practices

Most developers spend hours rewriting the same structure again and again.

So I built the **Oracle Database Development Handbook** as a Claude Skill.

Now I simply describe the object I need, and Claude generates Oracle SQL & PL/SQL code that already follows enterprise development standards.

# 💡 How It Works

## Step 1

Invoke the Claude Skill

```text
/oracle-db-development-handbook
```

---

## Step 2

Describe what you want to build.

Example:

```text
Create a procedure to insert records into EMP_MASTER
(emp_id, emp_name).
```

---

## Step 3

Claude automatically generates:

✅ Oracle Procedure

✅ Standard Documentation Header

✅ Company Coding Standards

✅ Naming Conventions

✅ Input Validations

✅ Exception Handling

✅ Comments

✅ Best Practices

Ready for code review and deployment.

# 🚀 Installation

1. Download this repository.

2. Open Claude Desktop.

3. Navigate to **Skills**.

4. Create a new Skill.

5. Upload:

```text
SKILL.md
sql.md
```

6. Save the Skill.

You're now ready to generate production-ready Oracle SQL & PL/SQL code.
